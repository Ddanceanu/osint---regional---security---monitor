# Project Scope

## Working title
OSINT Regional Security Monitor

## Project type
Personal software project focused on OSINT-based analytical support.

## General objective
The project aims to build a structured OSIND platform that collects and organizes public information relevant to regional security and international relations, with Romania as the main reference point.

## Analytical focus
The system is centered on developments involving Romania in relation to the Republic of Moldova, Ukraine, and Russia, in the broader of the Russia-Ukraine war and the institutional responses of NATO and the European Union regarding the eastern flank.

## MVP objective
The first version of the project will collect a controlled set of public documents from selected official sources and think tanks, process them into structured records, classify them by theme, extract key entities, and support basic analytical exploration through filters, trends, and short summaries.

## Why this project 
Regional security analysis depends on the ability to move from scattered public information to structured, interpretable output. This project focuses on that transition by combining source collection, document processing, thematic classification, entity extraction, and analytical exploration in a single workflow.

## Intended use
The platform is intended as a small-scale analytical support tool for monitoring developments related to mentioned countries using public and traceable sources.
